# LightShot
A game made for the Brackeys Game Jam inspired by Mario with a small twist added for the theme **Light**.

## Objective:
> Reach the door at the end of the level avoiding all enemies and weapons

## Controls:
- AD or Arrow Keys to move
- SPACE to Jump
- Q to Shoot
- E to Change Gravity
